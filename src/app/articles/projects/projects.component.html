Trello

Aplicación web para gestión de proyectos desarrollada con un enfoque full-stack. Incluye tableros, listas y tarjetas para organizar tareas, colaboración en tiempo real, asignación de usuarios, y funciones como etiquetas, comentarios y fechas límite. Implementada con Angular y ASP.NET Core.

Kahoot

Aplicación web interactiva para crear y jugar cuestionarios en tiempo real. Permite a los usuarios diseñar preguntas personalizadas, unirse a partidas mediante códigos únicos y competir con otros participantes. Desarrollada con Angular y Node.

Instagram

Aplicación web para compartir y explorar contenido multimedia. Incluye funciones como publicaciones con fotos y descripciones, comentarios, likes, perfiles de usuario, y un feed dinámico. Desarrollada con Angular y ASP.NET Core.

Penca de Fútbol

Plataforma web para participar en predicciones de resultados de partidos de fútbol. Incluye creación de ligas privadas, registro de predicciones, cálculo automático de puntajes y clasificación. Desarrollé el frontend en Angular y el backend en Node.

Algoritmos genéticos

Desarrollé un juego donde los players deben llegar a una meta esquivando obstáculos, inspirado en El juego más difícil del mundo. Utilicé algoritmos genéticos para entrenar a los players. Las decisiones de movimiento (adelante, atrás, izquierda, derecha) se basan en un "cerebro" evolucionado a través de generaciones. Implementé el proyecto en Unity con C#.

Mentimeter

Aplicación web interactiva para crear encuestas y presentaciones en tiempo real. Permite a los usuarios diseñar preguntas, recopilar respuestas del público mediante códigos únicos y visualizar resultados en gráficos dinámicos. Desarrollé el frontend en Angular y el backend en ASP.NET Core.
